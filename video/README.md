# Remotion Video Project

This project contains the programmatic video assets for the portfolio.

## Setup
Run `npm install` in this directory if you haven't already.

## Commands
- **Preview**: `npm start` - Opens the Remotion Player to preview videos.
- **Render**: `npm run render <composition-id>` - Renders a video to MP4.
  - Example: `npm run render HeroLoop`

## Compositions
- **HeroLoop**: Main landing page loop (10s).
- **TacticalVision**: Project 1 micro-video.
- **DecisionIntelligence**: Project 2 micro-video.
- **ScoutingEngine**: Project 3 micro-video.
